# AI Fake News Detector

An AI-powered fake news detection web application built using Flask, Machine Learning, OCR, and NewsAPI.

## Features

- Detects potentially fake or real news content
- Supports both text and image-based news verification
- OCR extraction from uploaded screenshots/images
- Confidence score analysis
- Modern responsive UI
- NewsAPI cross-reference support

## Technologies Used

- Python
- Flask
- Scikit-learn
- NLP (TF-IDF Vectorization)
- OCR (Tesseract)
- HTML/CSS/JavaScript
- NewsAPI
